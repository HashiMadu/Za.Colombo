## 🔗 作品リンク

## 📍 実店舗情報（架空）  
名古屋市栄｜スリランカ家庭料理

- 大須観音駅から135m  
- 代表的なスリランカライス&カレー  
- スリランカ人オーナーが経営するレストランで、本場スリランカ料理の味をお届けします。  
- スリランカ特有のスパイスとレシピを使って丁寧に作られている。
